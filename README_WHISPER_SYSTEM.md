## 🛠️ Installation

### Quick Setup

```bash
# Clone the repository
git clone <your-repo-url>
cd AiTSwarmX

# Install Whisper System dependencies
pip install -r requirements_whisper.txt

# Install Pylint for code validation
pip install pylint>=2.17.0

# Install async SMTP support
pip install aiosmtplib>=2.0.0

# Run the Whisper System
python whisper_core.py
```
